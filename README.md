# Kit Z80

 Este projeto tem como partida uma solicitação de um dos leitores do meu blog ([https://dqsoft.blogspot.com](https://youtube.com/@DQSoft)). Ele possui um kit dos anos 80, o CEDM-80, e gostaria de um firmware novo para ele.
 
 Para escrever o firmware o primeiro passo é ter o hardware e aí está o projeto: construir um microcomputador compatível com o CEDM-80 (versão 1, para ser mais preciso). É um projeto de médio a longo prazo, ficarei muito feliz se o concluir até a ~~metade do ano atual (2024)~~ o final do primeiro trimestre de 2026.
 
 O projeto está sendo documentado no blog e no meu canal no Youtube ([https://youtube.com/@DQSoft](https://youtube.com/@DQSoft))
 
 ## Testes do Circuito de Reset (02/06/24)
 
 Em Circuito/TesteReset está o esquema do circuito montado para testar três opções para o Reset.
 
 ## Teste do Uso do 74HC373 como Latch/Porta de Saída (13/06/24)
 
 Em Circuito/Teste373 está o esquema do circuito montado para testar o uso do 74HC373 como porta de saída.
 
 Em Programas/TstBlink está o programa usado no teste.
  
 ## Teste da  RAM (24/08/24)
 
 Em Programas/MTest está o programa, que usa o gerador de números pseudo-aleatorios do Pitfall para testar a memória RAM!
 
 ## Teste Inicial do Display (14/04/25)
 
 Com a montagem da placa de I/O iniciada, é hora de testar o circuito de acionamento do display e validar o valor dos resistores de limitação de corrente.
 
 O programa de testes está em Programas/TstDisp.
 
 ## Teste Final do Display (14/02/25)
 
 Após mais uma longa pausa, a montagem do circuito do display está completa.
 
 O programa de testes está em Programas/TstDisp.
